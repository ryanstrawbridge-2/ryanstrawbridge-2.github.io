How exact percentages work

The site already accepts CSS object-position values. The dropdown ships the 9 standard keywords. If you want pixel-precise placement, paste the exact percentage value (e.g., 37% 22%) into a project's heroFocalPoint field after editing the markdown directly, or ask Claude to extend the field to accept free-form strings.
